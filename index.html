object-properties

Shortcuts to Object.defineProperty and Object.defineProperties

Examples

 
import { defineGetter } from 'object-properties'; // or var defineGetter = require('object-properties').defineGetter;

let object = {};

defineGetter(object, 'test', () => 'test');
